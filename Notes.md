What are React Hooks used for?
 hooks allow React function style components to be able to do everything that a class style component can do and more


Note that hooks can only be used in function style components and must be called at the top scope of the function. That means a hook cannot be called inside of a loop or conditional. This restriction ensures that hooks are always called in the same order when a component is rendered.


What is the useEffect hook used for?

The hook provides a mechanism for handling asynchronous operations, subscriptions, and other non-trivial tasks while ensuring proper cleanup to avoid memory leaks or resource leaks.
